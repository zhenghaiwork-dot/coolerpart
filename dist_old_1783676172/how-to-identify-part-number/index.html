How to Identify Your Part

Follow this guide to identify your replacement part correctly — interface type, dimensions, and key specs.

Step 1: Determine the System

Which part of the cooler does your component belong to?

  • Water System — pumps, valves, filters, plumbing
  • Evaporation System — cooling pads, distributors, anti-clogging
  • Air System — fans, blowers, grilles, centrifugal parts
  • Electrical & Controls — motors, capacitors, control boards
  • Structure & Mounting — panels, brackets, seals, casters
  • Maintenance & Consumables — filters, wear parts, cleaning items

Step 2: Identify the Interface Type

How does the part connect to your cooler?

  • Threaded connection — note thread size (e.g., 1/2" NPT, M12)
  • Flange mount — note bolt pattern and hole diameter
  • Push-fit / quick connect — note tube diameter
  • Bracket/clamp mount — note bracket dimensions and hole spacing

Step 3: Measure Key Dimensions

  • For pumps: inlet/outlet size, overall dimensions, mounting hole pattern
  • For motors: shaft diameter, shaft length, mounting type (B3/B5/B14), frame size
  • For cooling pads: width x height x thickness
  • For filters: outer dimensions, thickness, frame type

Step 4: Note Electrical Parameters

If your part is electrical, record:

  • Voltage (V) and Frequency (Hz)
  • Power (W) or Current (A)
  • RPM (for motors)
  • Capacitor value if applicable

Step 5: Take Photos

Clear photos are the fastest way to get accurate identification:

  • Front view showing the whole part
  • Close-up of the interface/connection point
  • Any labels, part numbers, or model numbers
  • Scale reference (ruler or coin) next to the part
